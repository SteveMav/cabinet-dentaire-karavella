# Cabinet Dentaire Karavella - kit de prospection

Site statique multipage cree comme proposition non officielle pour Cabinet Dentaire Karavella a Gombe, Kinshasa.

## Pages

- `index.html` : accueil, positionnement, preuve, services et proposition de digitalisation.
- `soins.html` : services dentaires et informations patients.
- `espace-patient.html` : concept de portail patient securise.
- `rendez-vous.html` : formulaire qui prepare une demande WhatsApp.
- `contact.html` : coordonnees, horaires et acces.

## Notes

Le dossier `mailsender/` est ignore par Git, car il contient les outils d'emailing et les futures variables SMTP.
